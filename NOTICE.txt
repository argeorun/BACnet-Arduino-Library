================================================================================
BACNET PROTOCOL STACK
================================================================================

This library incorporates the BACnet Protocol Stack (bacnet-stack) located in
the src/bacnet/ directory:

BACnet Protocol Stack
Copyright (C) 2012 Steve Karg <skarg@users.sourceforge.net>
Project: https://github.com/bacnet-stack/bacnet-stack
License: GNU GPL v2 or later with linking exception

The bacnet-stack project was created and is maintained by Steve Karg. It
provides a complete, robust implementation of the BACnet protocol (ANSI/ASHRAE
Standard 135) for building automation and control networks.
